Inicio

Esta es una breve explicación de cómo funciona mi página web.

<section>

La etiqueta <section> se utiliza para agrupar contenido temático dentro de un documento HTML. Puede contener encabezados, párrafos, listas, entre otros elementos.

<article>

La etiqueta <article> se usa para representar contenido independiente y autónomo, como entradas de blog, noticias, comentarios, etc. Dentro de un <article> se pueden incluir elementos como encabezados, párrafos, imágenes, etc.

<aside>

La etiqueta <aside> se emplea para contenido relacionado o complementario al contenido principal de la página. Puede contener elementos como anuncios, enlaces a contenido relacionado, widgets, etc.

<fomr >

La etiqueta <from > representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos. Ejemplos comunes de secciones de navegación son menús, tablas de contenido e índices...

<header>

La etiqueta <header> se usa para Definir el encabezado de una pagina web , en el encabezados, puede contener un logo, un formulario de búsqueda, un nombre de autor y otros componentes..

<nav>

La etiqueta <nav> representa una sección de una página cuyo propósito es proporcionar enlaces de navegación, ya sea dentro del documento actual o a otros documentos. Ejemplos comunes de secciones de navegación son menús, tablas de contenido e índices...

<footer>

La etiqueta <footer> se emplea para el pie de página de la página web. Generalmente se utiliza para incluir información de contacto, enlaces útiles, derechos de autor, etc.

Más Etiquetas HTML

  • <time>: Especifica una fecha y/u hora.
  • <video>: Inserta un video en la página.
  • <audio>: Inserta audio en la página.
  • <input>: Crea un campo de entrada en un formulario.
  • <textarea>: Crea un área de texto multilínea en un formulario.
  • <select> y <option>: Crean listas desplegables en un formulario.
  • <table>, <tr>, <td>: Crean tablas y sus elementos.
